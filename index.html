Safe SomaFM

A defensive Home Assistant custom integration for listening to SomaFM with a local player, compact dashboard card, and sidebar launcher.

What it does

  • Runs a local Home Assistant browser player at /safe_somafm/player.
  • Provides a compact Lovelace card through /safe_somafm/card.js.
  • Adds a Safe SomaFM sidebar launcher that opens the full player in a new browser tab.
  • Supports station artwork, metadata, quality selection and automatic reconnect.

Dashboard card

Add /safe_somafm/card.js as a JavaScript module resource, then use:

type: custom:safe-somafm-card

Sidebar launcher

The Home Assistant sidebar entry opens the full player in a new tab. If the browser blocks it, a fallback button is shown.

Security posture

No credentials, no secrets, no arbitrary stream URL input, and no arbitrary proxy. SomaFM URLs are validated before use.

Safe SomaFM is not affiliated with SomaFM or Home Assistant.
